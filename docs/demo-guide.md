# LumeraKai Health - Demo Guide
*How to showcase your revolutionary healthcare AI platform*

## 🎯 Demo Objectives
- Demonstrate multi-agent AI coordination
- Show real healthcare problem solving
- Highlight competitive advantages
- Generate partnership interest

## 🚀 Quick Start Demo (5 minutes)

### Step 1: Start Local Demo
```bash
cd /Users/RoutWorx/lumerakai-health
./start-dev.sh
# Wait for services to start
open http://localhost:3000
```

### Step 2: Navigate to Agent Dashboard
- Click "Agent Dashboard" or go to `/dashboard`
- Show the clean, professional interface
- Point out real-time agent status indicators

### Step 3: Run Maria's Scenario
1. **Select "Maria's Scheduling Conflict"**
2. **Watch agents activate** (status lights turn green/yellow)
3. **Narrate the coordination:**
   - "Maria reports dental surgery conflict"
   - "Lumera-Admin detects chiropractor same day"
   - "Lumera-Doc checks medication interactions"
   - "Lumera-Pharmacy recommends safer alternatives"
   - "Lumera-Nurse coordinates rescheduling"

## 🎪 Full Demo Script (15 minutes)

### Opening Hook (2 minutes)
> "Healthcare coordination costs the US $200 billion annually in inefficiencies. 
> What if AI could listen to patient, family, and care team conversations 
> and prevent conflicts before they happen? Let me show you LumeraKai Health."

### Problem Statement (3 minutes)
**Show the pain points:**
- "Maria has diabetes and needs dental surgery"
- "She also has a chiropractor appointment the same day"
- "Pain medications could affect her blood sugar"
- "Without coordination, this becomes a medical crisis"

### Solution Demo (8 minutes)

#### Part 1: Agent Introduction (2 minutes)
**Show agent dashboard:**
- "Meet your AI care team - 5 specialized agents"
- "Each has distinct healthcare expertise"
- "They work together like a real care team"

#### Part 2: Live Scenario (4 minutes)
**Run Maria's scenario:**
1. **Patient Input**: "I can't come Thursday; I have dental surgery"
2. **Agent Activation**: Watch status lights change
3. **Real-time Coordination**: Agents communicate
4. **Problem Resolution**: Conflicts prevented, care optimized

#### Part 3: Interactive Elements (2 minutes)
**Show advanced features:**
- Type custom message: "What if Maria is allergic to acetaminophen?"
- Watch agents adapt and provide alternatives
- Demonstrate voice input capability

### Impact & Results (2 minutes)
**Highlight outcomes:**
- "Prevented medication conflict"
- "Optimized appointment scheduling"
- "Reduced family anxiety"
- "Saved 2+ hours of coordination time"
